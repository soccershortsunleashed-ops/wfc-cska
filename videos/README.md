# 🎬 Видео-демонстрация сайта ЖФК ЦСКА

## Полное демо

### `demo-full.webm`
Полная демонстрация всех функций и анимаций сайта (~2-3 минуты):
- Header и навигация
- Floating Dock с соцсетями
- Hero секция
- Карточки матчей с hover-эффектами
- Секция новостей
- Карусель игроков
- Секция спонсоров
- Footer
- Страница команды с фильтрацией
- Профиль игрока

---

## Короткие демо (папка `demos/`)

Каждое видео демонстрирует конкретную функцию (10-30 секунд):

### 1. Header и навигация
**Файл:** `01-header-navigation.webm`

**Демонстрирует:**
- Градиентный фон Header
- Логотип ЦСКА
- Навигационное меню
- Hover-эффекты на пунктах меню
- Адаптивный дизайн

---

### 2. Floating Dock (Соцсети)
**Файл:** `02-floating-dock-social.webm`

**Демонстрирует:**
- Плавающая панель социальных сетей
- Анимация увеличения при наведении (дуга вниз)
- Тултипы с названиями соцсетей
- Прозрачный фон с размытием (glass-morphism)
- Белые иконки на синем фоне

---

### 3. Карточки матчей
**Файл:** `03-match-cards-hover.webm`

**Демонстрирует:**
- Карточка ближайшего матча (синий градиент)
- Карточка последнего матча (с результатом)
- Эффект "парения" при наведении (hover)
- Логотипы команд (64-80px)
- Кнопки "Купить билет" и "Подробнее"
- Адаптивный дизайн

---

### 4. Карусель игроков
**Файл:** `04-players-carousel.webm`

**Демонстрирует:**
- Автоматическая прокрутка карусели
- Фотографии игроков
- Навигация стрелками
- Плавные переходы между слайдами
- Адаптивный дизайн

---

### 5. Фильтрация игроков
**Файл:** `05-player-filters.webm`

**Демонстрирует:**
- Фильтр по позиции (Вратари, Защитники, Полузащитники, Нападающие)
- Фильтр по составу (Основной, Молодёжный, Юниорский)
- Мгновенная фильтрация без перезагрузки страницы
- Счётчик найденных игроков
- Плавная анимация появления карточек

---

### 6. Поиск игроков
**Файл:** `06-player-search.webm`

**Демонстрирует:**
- Поле поиска по имени игрока
- Регистронезависимый поиск
- Поиск по частичному совпадению
- Мгновенное обновление результатов
- Очистка поиска

---

### 7. Карточки игроков с hover
**Файл:** `07-player-cards-hover.webm`

**Демонстрирует:**
- Карточки игроков с фотографиями
- Номер, имя, позиция
- Статистика (матчи, голы, передачи)
- Hover-эффекты (увеличение, тень)
- Клик для открытия модального окна

---

### 8. Профиль игрока
**Файл:** `08-player-profile.webm`

**Демонстрирует:**
- Полноэкранный градиентный фон
- Striped pattern (диагональные полосы)
- Крупное фото игрока (300px)
- Номер и имя крупным шрифтом
- Основная информация (рост, вес, дата рождения, гражданство)
- Детальная статистика с glass-morphism эффектами
- Голы выделены красным цветом ЦСКА
- Кнопка "Назад к составу"

## 🎥 Как использовать видео

### Для презентации руководству:
1. Покажите **полное демо** (`demo-full.webm`) для общего впечатления
2. Используйте **короткие демо** для детального объяснения конкретных функций
3. Акцентируйте внимание на:
   - Современный дизайн и анимации
   - Интерактивность и удобство
   - Быстрая работа без перезагрузок
   - Адаптивность для всех устройств

### Для документации:
- Встройте видео в презентацию PowerPoint/Keynote
- Добавьте ссылки в README.md
- Используйте для обучения команды

### Для социальных сетей:
- Короткие демо идеальны для Instagram/TikTok
- Полное демо для YouTube
- GIF-анимации из коротких видео для Twitter

## 📊 Технические детали

### Параметры записи:
- **Разрешение:** 1920x1080 (Full HD)
- **Формат:** WebM (VP9 codec)
- **FPS:** 30
- **Браузер:** Chromium (Playwright)

### Размеры файлов:
- Полное демо: ~10-20 MB
- Короткие демо: ~1-3 MB каждое

## 🔄 Перезапись видео

Если нужно перезаписать видео:

```bash
# Полное демо
node scripts/record-demo-video.mjs

# Короткие демо
node scripts/record-short-demos.mjs
```

**Примечание:** Убедитесь, что dev-сервер запущен на `http://localhost:3000`

---

## 📝 Примечания

1. **Случайные имена файлов:** Playwright генерирует случайные имена для видео. После записи переименуйте файлы в соответствии с порядком.

2. **Качество:** Для лучшего качества записывайте с `headless: false` (браузер видимый).

3. **Производительность:** Запись видео требует ресурсов. Закройте другие приложения для плавной записи.

4. **Редактирование:** Используйте видеоредакторы (DaVinci Resolve, Adobe Premiere) для:
   - Обрезки лишних кадров
   - Добавления титров и аннотаций
   - Наложения музыки
   - Конвертации в другие форматы (MP4, GIF)
