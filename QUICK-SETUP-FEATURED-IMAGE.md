# 🚀 Быстрая установка изображения главной новости

## 📋 Что нужно сделать (3 простых шага)

### Шаг 1: Сохраните изображение
Сохраните изображение с турниром "БРИДЖ ЦСКА" как:
```
featured-news-bridge-cska.jpg
```

### Шаг 2: Поместите в папку
Скопируйте файл в:
```
wfc-cska/public/seed-assets/featured-news-bridge-cska.jpg
```

### Шаг 3: Запустите команду
```bash
cd wfc-cska
npm run update-featured-news
```

## ✅ Готово!

После выполнения команды:
- ✅ Изображение установлено
- ✅ Новость создана/обновлена
- ✅ Данные в базе обновлены

Откройте http://localhost:3000 и увидите новое изображение на главной странице!

## ⚠️ Если возникли проблемы

### Изображение не найдено?
Проверьте путь:
```bash
ls wfc-cska/public/seed-assets/featured-news-bridge-cska.jpg
```

### Ошибка при запуске скрипта?
Убедитесь, что вы в правильной папке:
```bash
cd wfc-cska
pwd  # Должно показать путь к wfc-cska
```

### База данных не обновляется?
Перезапустите dev сервер:
```bash
npm run dev
```

## 💡 Альтернативный способ

Если не хотите использовать скрипт, можно:
1. Поместить изображение в `public/seed-assets/`
2. Вручную обновить новость через админку (когда будет готова)
