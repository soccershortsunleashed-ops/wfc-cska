## Запуск проекта

```bash
# Перейти в папку проекта
cd wfc-cska

# Запустить dev сервер
npm run dev
```

Откройте http://localhost:3000 в браузере.

## 🛠️ Частые задачи

### Обновить главную новость
```bash
npm run update-featured-news
```

### Извлечь данные с сайта
```bash
npm run setup-from-site
```

### Открыть базу данных
```bash
npx prisma studio
```

### Собрать проект
```bash
npm run build
npm run start
```

## 🆘 Помощь

### Проблемы с изображениями
1. Проверьте путь к файлу
2. Очистите кэш браузера (Ctrl+F5)
3. Проверьте размер: 1200x600px для главной новости

### Проблемы с базой данных
```bash
# Пересоздать базу
npx prisma migrate reset

# Заполнить данными
npm run seed-extracted
```

### Проблемы с зависимостями
```bash
# Переустановить пакеты
rm -rf node_modules package-lock.json
npm install
```
