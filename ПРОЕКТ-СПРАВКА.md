## 🎨 Цветовая схема

### Основные цвета ЦСКА
- **Синий**: `#0033A0` (primary)
- **Красный**: `#E4002B` (accent)
- **Белый**: `#FFFFFF`

### Применение в Tailwind
```tsx
// Синий ЦСКА
className="bg-[#0033A0] text-white"

// Красный ЦСКА
className="bg-[#E4002B] text-white"

// Градиенты
className="bg-gradient-to-r from-[#0033A0] to-[#E4002B]"
```

## 🗄️ База данных

### Prisma
```bash
# Открыть Prisma Studio
npx prisma studio

# Применить миграции
npx prisma migrate dev

# Сгенерировать клиент
npx prisma generate
```

### Схема
- **Players**: игроки команды
- **Matches**: матчи
- **News**: новости
- **Sponsors**: спонсоры

Файл: `prisma/schema.prisma`

## 📦 Скрипты

### Разработка
```bash
npm run dev          # Запуск dev сервера
npm run build        # Сборка продакшн
npm run start        # Запуск продакшн сервера
```

### Данные
```bash
npm run extract-seed              # Извлечь данные с сайта
npm run extract-matches           # Извлечь матчи
npm run copy-assets              # Скопировать изображения
npm run seed-extracted           # Заполнить БД
npm run setup-from-site          # Полная настройка
npm run update-featured-news     # Обновить главную новость
```
