## 📊 Структура данных для расширения

### Текущая модель Match:
```prisma
model Match {
  id              String    @id @default(cuid())
  type            MatchType
  opponentName    String
  opponentLogoUrl String?
  cskaLogoUrl     String?
  matchDate       DateTime
  venue           String?
  scoreHome       Int?
  scoreAway       Int?
  tournament      String?
  createdAt       DateTime  @default(now())
  updatedAt       DateTime  @updatedAt
}
```

### Предлагаемые дополнения:
- `slug` - для SEO-friendly URL
- `season` - сезон (например, "2024/2025")
- `round` - тур (например, "Тур 15")
- `status` - статус матча (SCHEDULED, LIVE, FINISHED, POSTPONED, CANCELLED)
- `stadium` - название стадиона
- `attendance` - количество зрителей
- `referee` - главный судья
- `highlights` - ссылка на видеообзор
- `description` - описание матча
- `isHome` - домашний/выездной матч
