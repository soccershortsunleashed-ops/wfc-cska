# Цветовая схема ЦСКА

## Основные цвета бренда

### Primary - Синий ЦСКА
- **HEX**: `#143d8d`
- **OKLCH**: `oklch(0.35 0.15 264)`
- **Использование**: Основной цвет бренда, кнопки, ссылки, акценты
- **Контрастность с белым**: 8.5:1 ✅ (AAA)
- **Контрастность с черным**: 2.5:1 ✅ (AA для крупного текста)

### Accent - Красный ЦСКА
- **HEX**: `#e30613`
- **OKLCH**: `oklch(0.55 0.25 25)`
- **Использование**: Call-to-action, важные элементы, предупреждения
- **Контрастность с белым**: 4.8:1 ✅ (AA)
- **Контрастность с черным**: 4.4:1 ✅ (AA)

### Secondary - Золотой ЦСКА
- **HEX**: `#d4af37`
- **OKLCH**: `oklch(0.75 0.12 85)`
- **Использование**: Вторичные акценты, награды, достижения
- **Контрастность с белым**: 1.8:1 ⚠️ (Только для декора)
- **Контрастность с темным текстом**: 7.2:1 ✅ (AAA)

## Рекомендации по использованию

### ✅ Правильно
- Синий для основных кнопок и ссылок
- Красный для важных действий (купить билет, подписаться)
- Золотой для наград, достижений, премиум-контента
- Белый текст на синем/красном фоне
- Темный текст на золотом фоне

### ❌ Избегать
- Золотой текст на белом фоне (низкая контрастность)
- Красный текст на синем фоне (плохая читаемость)
- Слишком много красного (агрессивно)
- Черный текст на темно-синем (низкая контрастность)

## Проверка контрастности

### WCAG 2.1 Уровни
- **AAA**: Контрастность ≥ 7:1 (обычный текст), ≥ 4.5:1 (крупный текст)
- **AA**: Контрастность ≥ 4.5:1 (обычный текст), ≥ 3:1 (крупный текст)

## Использование в коде

### Tailwind CSS классы
```tsx
// Primary (синий)
<button className="bg-primary text-primary-foreground">
  Кнопка
</button>

// Accent (красный)
<button className="bg-accent text-accent-foreground">
  Купить билет
</button>

// Secondary (золотой)
<div className="bg-secondary text-secondary-foreground">
  Премиум контент
</div>

// Границы с брендовым оттенком
<input className="border-border focus:ring-ring" />
```

### CSS переменные
```css
/* Прямое использование брендовых цветов */
.custom-element {
  background: var(--cska-blue);
  color: white;
}

/* Использование семантических токенов */
.button-primary {
  background: var(--primary);
  color: var(--primary-foreground);
}
```
