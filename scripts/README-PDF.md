# Генерация PDF документов

## Описание

Скрипт `generate-pdf-v2.mjs` конвертирует Markdown документы в профессиональные PDF файлы для отправки руководству.

## Возможности

- ✅ Полная поддержка кириллицы
- ✅ Замена относительных ссылок на полные GitHub URL
- ✅ Профессиональное форматирование с цветами ЦСКА
- ✅ Красивая типографика (шрифт Inter)
- ✅ Правильные разрывы страниц
- ✅ Оптимизация для печати

## Использование

```bash
# Генерация PDF файлов
node scripts/generate-pdf-v2.mjs
```

## Результат

Скрипт создаёт 2 PDF файла в корне проекта:
- `ПРЕЗЕНТАЦИЯ-НОВОГО-САЙТА.pdf` - полная презентация (60+ страниц)
- `ПИСЬМО-РУКОВОДСТВУ.pdf` - краткое письмо (10-15 страниц)

## Технологии

- **Puppeteer** - генерация PDF через headless Chrome
- **markdown-it** - парсинг Markdown
- **Google Fonts (Inter)** - профессиональный шрифт

## Стили

### Цвета ЦСКА
- Синий: `#0033A0` - заголовки и ссылки
- Красный: `#E4002B` - акценты и подчёркивания

### Типографика
- Заголовок H1: 28pt, жирный, с красной линией снизу
- Заголовок H2: 20pt, с серой линией сверху
- Заголовок H3: 16pt
- Основной текст: 11pt, выравнивание по ширине

### Таблицы
- Заголовки: синий фон, белый текст
- Чётные строки: светло-серый фон
- Границы: светло-серые

## Примечания

- Все изображения и видео ссылаются на GitHub
- PDF оптимизированы для печати на A4
- Поля: 20mm со всех сторон
- Автоматические разрывы страниц
