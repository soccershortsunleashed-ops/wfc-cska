## 📋 Что было исправлено

### 1. Логика отображения команд в матчах
**Проблема:** ЦСКА всегда отображался слева, независимо от того, играет команда дома или в гостях.

**Решение:** Реализована правильная логика:
- Команда, играющая **дома** → всегда **слева**
- Команда, играющая **в гостях** → всегда **справа**

**Затронутые файлы:**
- `components/matches/match-list-item.tsx` - список матчей
- `components/matches/match-detail.tsx` - детальная страница матча

### 2. Улучшение дизайна шапки карточки матча
**Проблема:** Шапка выглядела невзрачно, текст был мелким и раскидан по всей ширине.

**Решение:**
- Все элементы размещены в одной строке
- Увеличены размеры шрифтов (text-base → text-lg)
- Добавлены визуальные разделители
- Улучшена компактность и читаемость

**Файл:** `components/matches/match-detail.tsx`

### 3. Исправление ошибки гидратации React
**Проблема:** Ошибка "Hydration mismatch" в компоненте NavigationMenuTrigger.

**Решение:** Удален лишний пробел `{" "}` между children и иконкой.

**Файл:** `components/ui/navigation-menu.tsx`

### Логика расположения команд

#### До:
```tsx

```

#### После:
```tsx
// Проверка isHome
{match.isHome ? (
  // ЦСКА дома - слева
  <div>ЦСКА (Дома)</div>
  <div>Соперник (В гостях)</div>
) : (
  // ЦСКА в гостях - справа
  <div>Соперник (Дома)</div>
  <div>ЦСКА (В гостях)</div>
)}
```
