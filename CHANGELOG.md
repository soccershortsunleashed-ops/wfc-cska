## [1.1.0] - 2026-01-25

### Добавлено
- 🌈 **Система градиентов ЦСКА** - вдохновлена дизайном ФК "Локомотив"
  - 8 типов градиентов (hero, subtle, card, button, accent, hover, text, animated)
  - Полная палитра цветов (50-900 оттенков) для синего и красного
  - Тестовая страница `/test-gradients` для демонстрации
- 📚 **Документация по градиентам**
  - `COLOR-SCHEME-ENHANCED.md` - расширенная цветовая палитра
  - `GRADIENTS-QUICK-GUIDE.md` - быстрый справочник
  - `GRADIENT-IMPLEMENTATION-SUMMARY.md` - итоги внедрения

### Изменено
- 🎨 **Hero секция** - добавлен градиентный оверлей (синий → красный)
- 🎨 **Заголовок "ЦСКА"** - применен градиентный текст
- 🔴 **Кнопка CTA** - красный градиент с эффектом hover
- 🔵 **Header** - тонкий синий градиент фон
  - Белый текст навигации для контраста
  - Прозрачные hover состояния (white/10)
  - Синий фон выпадающих меню с белым текстом
- 🔵 **Footer** - тонкий синий градиент фон
  - Белый текст и границы
  - Белые иконки социальных сетей
- 🎯 **Social Dock (FloatingDock)**
  - Прозрачный фон с backdrop blur
  - Белые границы и иконки
  - Плавные hover эффекты
  - Консистентный дизайн на desktop и mobile

### Технические улучшения
- Обновлены CSS переменные для градиентов в `globals.css`
- Добавлены utility классы для быстрого применения градиентов
- Оптимизирована производительность градиентных анимаций
- Улучшена доступность с правильными цветовыми контрастами

## [1.0.0] - 2026-01-25

### Добавлено

#### Основные функции
- ⚽ Главная страница с hero секцией
- 👥 Страница игроков с фильтрами и поиском
- 📰 Секция последних новостей
- 🏆 Отображение ближайших и прошедших матчей
- 🎨 Карусель игроков с плавной анимацией
- 🤝 Секция спонсоров
- 🔗 FloatingDock для социальных сетей

#### UI/UX
- 📱 Адаптивный дизайн (mobile, tablet, desktop)
- 🎨 Фирменные цвета ЦСКА (#0033A0, #E4002B)
- ✨ Плавные анимации с Framer Motion
- 🖼️ Оптимизация изображений через Next.js Image
- ♿ Доступность (WCAG 2.1 AA)

#### Технические возможности
- 🔌 API Routes для данных игроков, новостей и матчей
- 🗄️ База данных SQLite с Prisma ORM
- 📊 Модальные окна с деталями игроков
- 🔍 Поиск игроков по имени
- 🎯 Фильтрация по позиции и команде
- 📈 Статистика игроков

#### Документация
- 📖 Подробный README.md
- 🚀 Руководство по деплою (DEPLOYMENT.md)
- 🎨 Документация по цветовой схеме (COLOR-SCHEME.md)
- 📝 Руководство по типографике (TYPOGRAPHY.md)
- 🔗 Руководство по FloatingDock (FLOATING-DOCK-GUIDE.md)
- 🔌 Документация по интеграции с Bitrix24

#### Инфраструктура
- 🔧 Настройка ESLint и TypeScript
- 📦 Конфигурация Prisma
- 🎨 Настройка Tailwind CSS v4
- 🔄 Скрипты для извлечения данных с сайта
- 🌱 Seed скрипты для заполнения БД

### Технологический стек
