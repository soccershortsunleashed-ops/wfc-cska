# ⚡ Быстрая публикация на GitHub

## 🚀 За 5 минут

### 1. Создайте репозиторий на GitHub

Перейдите на https://github.com/new и создайте новый репозиторий:

- **Repository name:** `wfc-cska`
- **Description:** `⚽ Официальный сайт ЖФК ЦСКА - Next.js 16, React 19, TypeScript, Tailwind CSS`
- **Visibility:** Public
- **НЕ** добавляйте README, .gitignore или LICENSE (они уже есть)

### 2. Выполните команды

```bash
cd wfc-cska

# Инициализация Git
git init

# Добавление всех файлов
git add .

# Первый коммит
git commit -m "feat: initial commit - WFC CSKA website"

# Добавление remote (замените yourusername на ваш GitHub username)
git remote add origin https://github.com/yourusername/wfc-cska.git

# Переименование ветки в main
git branch -M main

# Push на GitHub
git push -u origin main
```

### 3. Настройте репозиторий

На странице репозитория на GitHub:

1. **About** (справа вверху):
   - Description: `⚽ Официальный сайт ЖФК ЦСКА`
   - Website: `https://wfccska.ru`
   - Topics: `nextjs`, `react`, `typescript`, `tailwindcss`, `prisma`, `football`, `cska`

2. **Settings → General**:
   - ✅ Issues
   - ✅ Projects

### 4. Обновите README

Замените в `README.md`:
```
https://github.com/yourusername/wfc-cska
```
на ваш реальный URL.
