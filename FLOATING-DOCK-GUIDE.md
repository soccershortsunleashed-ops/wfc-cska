# 🎯 FloatingDock - Интерактивный док социальных сетей

## ✨ Что это?

FloatingDock - это стильный интерактивный компонент для отображения социальных сетей с эффектом увеличения иконок при наведении, как в macOS Dock.

## 🎨 Особенности

### Desktop версия
- **Эффект магнификации** - иконки плавно увеличиваются при приближении курсора
- **Всплывающие подсказки** - при наведении показывается название социальной сети
- **Плавные анимации** - используется Framer Motion для smooth transitions
- **Фирменные цвета** - иконки окрашены в цвета ЦСКА (синий и красный)

### Mobile версия
- **Компактная кнопка** - не занимает много места
- **Выпадающее меню** - иконки появляются при нажатии
- **Анимированное появление** - плавная анимация открытия/закрытия

## 📍 Где используется

1. **Header** (шапка сайта)
   - Расположение: правая часть навигации
   - Видимость: только на desktop (≥768px)

2. **Footer** (подвал сайта)
   - Расположение: под логотипом клуба
   - Видимость: на всех устройствах

## 🎮 Как использовать

### Просто наведите курсор!

1. Откройте сайт: http://localhost:3000
2. Найдите иконки социальных сетей в header или footer
3. Наведите курсор на иконки
4. Наслаждайтесь плавной анимацией увеличения!

### На мобильных устройствах

1. Нажмите на кнопку с иконкой меню
2. Иконки социальных сетей появятся вертикально
3. Нажмите на нужную иконку для перехода

## 🛠️ Технические детали

### Компоненты

```
components/
├── ui/
│   └── floating-dock.tsx      # Основной компонент
└── layout/
    └── social-dock.tsx        # Обертка для соц. сетей
```

### Зависимости

- `framer-motion` - анимации
- `@tabler/icons-react` - иконки
- `react-icons` - VK и Telegram иконки
- `lucide-react` - YouTube и Twitter иконки

### Параметры анимации

```typescript
// Размеры иконок
Обычный размер: 40px
Увеличенный размер: 80px

// Spring параметры
mass: 0.1
stiffness: 150
damping: 12

// Радиус влияния
150px от центра иконки
```

## 🎨 Кастомизация

### Изменить цвета иконок

Отредактируйте `components/layout/social-dock.tsx`:

```tsx
icon: (
  <FaVk className="h-full w-full text-[ВАШ-ЦВЕТ]" />
)
```

### Добавить новую социальную сеть

1. Импортируйте иконку
2. Добавьте объект в массив `socialLinks`:

```tsx
{
  title: "Instagram",
  icon: <Instagram className="h-full w-full text-[var(--cska-blue)]" />,
  href: "https://instagram.com/wfccska",
}
```

### Изменить размеры

Отредактируйте `components/ui/floating-dock.tsx`:

```tsx
// Измените значения в useTransform
let widthTransform = useTransform(distance, [-150, 0, 150], [40, 100, 40]);
//                                                            ↑   ↑    ↑
//                                                         мин макс  мин
```

## 💡 Советы

1. **Плавность анимации** зависит от производительности устройства
2. **Эффект увеличения** работает только на desktop
3. **Подсказки** появляются при наведении на 200ms
4. **Цвета** автоматически адаптируются к темной теме

## 🐛 Известные особенности

- На очень медленных устройствах анимация может быть менее плавной
- В Safari иногда требуется обновить страницу для корректной работы анимаций
